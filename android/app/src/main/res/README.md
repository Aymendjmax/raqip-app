# Android launcher icons (mipmap)

هذه المجلدات تعتمد آلية Android القياسية لاختيار الأيقونة حسب كثافة الشاشة:

- `mipmap-mdpi/ic_launcher.png` → أجهزة الكثافة المنخفضة/القياسية
- `mipmap-hdpi/ic_launcher.png` → أجهزة HDPI
- `mipmap-xhdpi/ic_launcher.png` → أجهزة XHDPI
- `mipmap-xxhdpi/ic_launcher.png` → أجهزة XXHDPI
- `mipmap-xxxhdpi/ic_launcher.png` → أجهزة XXXHDPI

لا تحتاج لاختيار يدوي داخل الكود؛ النظام يختار الأنسب تلقائيًا.
