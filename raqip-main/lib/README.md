# lib/

هذا المجلد هو قلب التطبيق (UI + logic).

## التقسيم المعتمد
- `main.dart`
  نقطة التشغيل، تحميل الإعدادات المحفوظة (اللغة/المظهر) ثم تشغيل التطبيق.
- `home_screen.dart`
  شاشة الخريطة الرئيسية، التوجيه، إدارة الحالة السفلية (Bottom Panel)، وتحديد الموقع.
- `saved_screen.dart`
  شاشة المحفوظات (إضافة/بحث/تعديل/نسخ/حذف/عرض على الخريطة).
- `settings_screen.dart`
  الإعدادات العامة + كلاس النصوص `S` (تعريب/ترجمة واجهة التطبيق).
- `theme.dart`
  إعدادات الثيم والألوان.

## ملاحظات تنظيمية
- أي نص جديد في الواجهة يجب إضافته في `S` داخل `settings_screen.dart`.
- منطق الخريطة والتوجيه يبقى في `home_screen.dart` فقط لتقليل التشتيت.
- نقطة التوقف الحالية: إزالة واجهة "أقرب محطة وقود" من البوكس السفلي مؤقتًا.
